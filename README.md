📍 GeoNotes
Flutter ve Spring Boot ile geliştirilmiş konum tabanlı not paylaşım mobil uygulaması.

GeoNotes, kullanıcıların konum tabanlı notlar oluşturabileceği, harita üzerinde içerik paylaşabileceği modern bir mobil uygulamadır.
Proje, **Flutter (Frontend)** ve **Spring Boot (Backend)** teknolojileri kullanılarak geliştirilmiştir.


 🚀 Özellikler:

* 📍 Konum bazlı not ekleme
* 🗺️ Google Maps entegrasyonu
* ☁️ Firebase ile veri yönetimi
* 📸 Görsel yükleme (image picker & storage)
* 📦 PostgreSQL veritabanı desteği
* 🐳 Docker ile container desteği
* 🔄 Jenkins ile CI/CD altyapısı


🛠️ Kullanılan Teknolojiler:
 
📱 Frontend (Mobile)
   * Flutter 
   * Firebase (Firestore, Storage) 
   * Google Maps API 
   * Geolocator

⚙️ Backend:
  * Java 21 
  * Spring Boot 
  * Spring Data JPA 
  * PostgreSQL 

🧪 Test & Araçlar:
   * JUnit & Mockito 
   * Selenium 

⚙️ DevOps:
   * Docker 
   * Jenkins 


📂 Proje Yapısı:

GeoNotes/
│
├── frontend/        # Flutter mobil uygulama
├── backend/         # Spring Boot REST API
├── docker-compose.yaml
└── README.md



⚙️ Kurulum:
  🔹 Backend (Spring Boot)
     cd backend
     ./mvnw spring-boot:run


 🔹 Frontend (Flutter)
     cd frontend
     flutter pub get
     flutter run


 🔹 Docker ile Çalıştırma
     docker-compose up --build


 🗄️ Veritabanı:
     * PostgreSQL kullanılmaktadır
     * Docker üzerinden otomatik ayağa kalkar 


🔐 Güvenlik:
  * Hassas bilgiler `.gitignore` ile korunmaktadır 
  * Ortam değişkenleri ile yönetilmesi önerilir


🎯 Amaç:
Bu proje, modern mobil uygulama geliştirme süreçlerinde:

* Full-stack mimari
* Harita tabanlı uygulama geliştirme
* DevOps süreçleri (Docker + Jenkins)

konularında pratik yapmak amacıyla geliştirilmiştir.
